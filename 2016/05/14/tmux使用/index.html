1 Tmux使用简单教程

1.1 简介

Tmux是unix下面的一个可以在终端窗口中运行多个终端会话的工具; 可以有效的帮我们提高工作效率…

1.2 MacOS下安装

1.3 常用功能

1.3.1 窗格操作

  • 水平分割窗格: C-b %
  • 垂直分割窗格: C-b "
  • 焦点移动: C-b 上下左右

1.3.2 窗口操作

  • 创建窗口: C-b c
  • 切换窗口: C-b <NUM> -> 窗口对应的数字;

1.3.3 会话

  • 在shell中创建会话: tmux new -s <name>
  • 在tmux中创建: C-b : new -s <name>
  • 获取会话列表: C-b ls
  • 切换会话: tmux attach -t <name>
  • 退出会话: tmux -d
  • 关闭会话: tmux kill-session -t <num>

1.3.4 高效的结对编程

将tmux会话地址,分享给其他人; 其他人通过ssh接入会话;

  1. 使用tmate工具
    • 安装: brew install tmate
    • 开启一个新的会话: tmate
