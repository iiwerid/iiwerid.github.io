Vimium使用

Veröffentlicht am 2016-01-19   |   in tool   |  

1 Vimium使用

一款工具, chrome插件, 可以使我们像操作vim一样使用快捷键操作chrome浏览器网页;

1.1 Vimium常用快捷键

1.1.1 页面滚动

  • j : 向下滚动;
  • k : 向上滚动;
  • gg : 到页面顶部;
  • G : 到页面底部;
  • d : 向下翻页;
  • u : 向上翻页;

1.1.2 打开网页

  • 在网页上看到喜欢的链接, 使用p或者P(新建网页)直接打开;
  • 从收藏夹和历史记录快捷打开: 按下o;

1.1.3 打开网页上任意一个链接

  1. 步骤:
    1. 按下f;
    2. 输入对应字母;

1.1.4 快速切换标签页

  • 按下T, 然后选择;
  • J :下一个标签页;
  • K : 上一个标签页;

1.1.5 关闭,恢复标签页

  • x : 关闭当前页面;
  • X : 恢复刚关闭的页面;

1.1.6 帮助

shift+/

1.1.7 其他

  • gs : 查看源代码;
  • yy : 复制当前url到剪贴板;
  • yf : 拷贝哦某个url到剪贴板;
  • gu : 跳转到父页面;
  • i : 输入模式; esc退出;
  • gi : 将光标定位到输入框, 使用tab切换;
