1.1 列表

可以加入checkbox，来标记任务的完成情况，可以根据子任务完成情况计算总任务进度;

  • M-S-RET，会输入一个自带checkbox的列表项；
  • C-c C-c改变checkbox状态；
  • M-left/right：改变列表层级关系；
  • M-up/down：上下移动列表项
  • [X] 完成合成宠物的算法改进； [3/3]
    1. [X] 分析目前合成宠物的算法;
    2. [X] 对需要改进的算法进行确认，分析；
    3. [X] 编码，测试进行改进；

有序：以1. 或者1) 开头 加 空格； 无序:以-，+，* 开头； 列表结束需要空俩行， 如果想加入同级别的可以M-RET

1.3 表格

输入表头：

Name num sub totle
Hello 123 90 213
llo 345 91 436

在#+TBLFM位置使用C-c C-c可以更新表格； 主要使用：使用|作为表格分隔， tab切换； 输入|- 然后按tab会产生分割线； tab/shift tab ; 对其表格C-c C-c; 在totle一列，的某个位置输入=$2+$3;然后C-u C-c C-c自动计算出totle 使用enter，跳到下一列； 使用M-上下左右移动行列，上下移动行，左右移动列； 使用M-S-上下左右，插入行列

1.4 链接

使用[[]], 中间空格；内容； www.batter 如果是其他格式，图片，pdf html，需要使用C-c C-x C-v 让其显示预览； [[http://xxxx][]]%EF%BC%9A 第二个[]中的内容，会显示出来；但是指向第一个链接内容；
