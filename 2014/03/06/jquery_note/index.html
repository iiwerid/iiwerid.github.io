JQuery学习

Veröffentlicht am 2014-03-06   |   in javascript   |  

1 JQuery学习

1.1 JQuery为什么会出现?

JQuery主要是为了解决javascript的部分不足;

1.1.1 javascript的弊端

  1. 复杂的文档对象模型DOM;
  2. 不一致的浏览器实现;
  3. 缺乏便捷的开发,调试工具;

1.1.2 JQuery优点

  1. 轻量级;
  2. 强大的选择器;
  3. 出色的DOM操作;
  4. 可靠的事件处理;
  5. 完善的兼容性;
  6. 链式操作;
  7. 不污染顶级变量:只建立jQuery对象;
  8. 丰富的插件;
  9. 完善的文档;
  10. 开源;

简化了遍历HTML文档, 操作dom, 处理事件, 执行动画, 开发Ajax的操作;

1.2 jQuery与dom

1.2.1 $(document).ready()

与window.onload区别;

  1. 时机:

    window.onload: 必须等网页所有内容加载完才执行,包括图片等; $(document).ready:所有dom结构绘制完毕后执行;

  2. 编写个数

    window.onload:只能编写一个; 多个会被覆盖; $(document):则可以多个, 并都会执行;

  3. 简化:

    $(document)可以简化: $(function() {…})

1.2.2 jQuery对象与Dom对象转换

  1. jQuery -> DOM
  2. DOM -> jQuery

1.2.3 jQuery与其他库冲突解决

  1. jQuery库后导入

    jQuery.noConflict(); //将变量$控制权, 让给其他库;

  2. jQuery先导入

    使用jQuery替代$,就可以了;

1.3 jQuery选择器

1.3.1 常用的CSS选择器

  1. 标签: td {…};
  2. id : #note {…};
  3. 类　: .note, div.note ;
  4. 群组: td,p,div,a {…}
  5. 后代选择器: #links a {}
  6. 通配选择器: * {..}

1.3.2 完善的处理机制

$('#tt').css("color", "red"); //当$('#tt')不存在时,不会报错;

判断某个元素是否存在:

  1. if ( $('#tt').length　> 0) ;
  2. if ( $('#tt')1 ) ; //转换dom对象判断

1.3.3 jQuery选择器

  1. 基本选择器同Css差不多
  2. 层次选择器
    1. $("ancestor descendant") : 后代元素;
    2. $("parent > child"): 子元素;
    3. $("prev+next"): 紧接在prev元素后的next元素; ==>$('prev').next('netx')
    4. $('prev~siblings'): prev元素 之后 的所有siblings元素; ==> $('prev').nextAll(…);
  3. 过滤选择器
    1. 基本过滤选择器
      1. :first; $("div:first") 选取第一个div元素;
      2. :last;
      3. :not(xxx);
      4. :even; 索引为偶数;
      5. :odd; 奇数;
      6. :eq(index);
      7. :gt(index), :lt(index);
      8. :header 选取网页中所有的标题
      9. :animated 当前正在执行动画的所有元素;
    2. 内容过滤选择器
      1. :contain(text);
      2. :empty 选取不包含子元素或文本的空元素;
      3. :has(selector);
      4. :parent 选取所有拥有子元素或文本的元素;
    3. 可见性过滤选择器
      1. :hidden;
      2. :visible;
    4. 属性过滤选择器
      1. [attribute]; 比如$('div[id]') 拥有属性id的元素;
      2. [attribute=val];
      3. [attribute!=val];
      4. [attribute^=val]; 选取属性的值以val开始的元素;
      5. [attribute$=val]; 结尾;
      6. [attribute*=val]; 包含;
      7. [selector][selector][selector] 复合属性;
    5. 子元素过滤选择器
      1. :nth-child(index/even/odd/equation); 选取每个父元素下索引匹配的元素;index从1开始;
      2. :first-child;
      3. :last-child;
      4. :only-child;
    6. 表单对象属性过滤选择器

      对所选择的 表单元素 进行过滤;

      1. :enabled ; 选取 表单内 所有可用元素;
      2. :disabled;
      3. :checked; 选取所有被选中的元素,单选框,复选框; 比如:$('input:checked');
      4. :selected; 选取所有被选中的选项元素,下拉列表; $('select :selected');
    7. 表单选择器
      1. :input 选取所有的<input>、<textarea>、<select>、<button>元素；
      2. ：text 所有单行文本框；
      3. :password 密码框;
      4. :radio 单选框;
      5. :checkbox
      6. :image
      7. :reset
      8. :button
      9. :file
      10. hidden
    8. 其他注意
      1. 选择器中包含., #, (, ]等特殊字符

        使用转义符进行转义; \\; 比如$('#id\\#b');

      2. 如果含有空格,意义不同

1.4 jQuery操作dom

1.4.1 jQuery中的dom操作

  1. 查找节点
  2. 创建节点
    1. 创建元素节点, 使用$(..); 比如:var $li1 = $("<li></li>"); $("ul").append($li1);
    2. 创建文本节点;同上; var $li1 = $("<li>苹果</li>");
  3. 插入节点
    1. append;
    2. appendTo; 与append函数的作用主体客体相反;
    3. prepend;
    4. prependTo;
    5. after;
    6. insertAfter;
    7. before;
    8. insertBefore;
  4. 删除节点
    1. remove(); 比如$("ul li:eq(1)").remove(); 或 $("ul li").remove("li[title!=苹果]");
    2. empty(); 清空节点, 不是删除节点;清空内容;
  5. clone(); 复制节点;
  6. 替换节点:replaceWith()
  7. 包裹节点:wrap, wrapAll, wrapInner
  8. 属性操作
    1. 获取,设置: attr("key"), attr("key", "val");
    2. 删除: removeAttr
  9. 样式操作
    1. 获取, 设置: attr("class", "xx");
    2. 追加; addClass("class", "yy");
    3. 移除; removeClass("xx yy"); 移除所有removeClass();
    4. 切换; toggle();
    5. 判断是否含有; hasClass;
  10. 设置和获取html,文本和值
    1. html(); 类似innerHTML, 用来读取或设置某个元素的html的内容;
    2. text() ==>innerText;
    3. val();
  11. 遍历
    1. children()
    2. next();
    3. prev();
    4. siblings(): 获取前后所有的同辈元素;
    5. closest()获取最近的匹配元素;
    6. 其他:find, filter, nextAll, prevAll, parent
  12. CSS-DOM操作
    1. css();
    2. offset()
    3. position()
    4. scrollTop, scrollLeft;
